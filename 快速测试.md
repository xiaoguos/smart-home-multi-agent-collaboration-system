# 🧪 Moss AI 聊天系统 - 快速测试指南

## ✅ 前置检查

在开始测试前，请确保已完成以下步骤：

1. ✅ 安装所有依赖（运行 `.\script\install_dependencies.ps1`）
2. ✅ 编译 Cangjie 后端（`cd app/backend && cjpm build`）
3. ✅ 启动所有服务

## 🚀 启动服务

### 自动启动（推荐）
```powershell
.\script\start_all_services.ps1
```

### 手动启动
打开 **3 个** PowerShell 终端窗口：

**终端 1 - Conductor Agent:**
```bash
cd agents/conductor_agent
python main.py --host localhost --port 12000
```

**终端 2 - Cangjie 后端:**
```bash
cd app/backend
dist\release\bin\main.exe
```

**终端 3 - 前端:**
```bash
cd app
pnpm dev
```

## 🐛 常见问题排查

### ❌ 错误：Conductor Agent 未响应

**检查清单：**
- [ ] 检查 Conductor Agent 是否正在运行
- [ ] 访问 `http://localhost:12000/` 是否有响应
- [ ] 查看 Conductor Agent 终端窗口的错误日志

**解决方法：**
```bash
# 重新启动 Conductor Agent
cd agents/conductor_agent
python main.py --host localhost --port 12000
```

### ❌ 错误：Cangjie 后端未响应

**检查清单：**
- [ ] 检查后端是否已编译
- [ ] 检查可执行文件是否存在于 `app/backend/dist/release/bin/main.exe`
- [ ] 查看后端终端窗口的错误日志

**解决方法：**
```bash
cd app/backend
cjpm build
dist\release\bin\main.exe
```

### ❌ 错误：前端显示连接错误

**检查清单：**
- [ ] 前端是否正确运行在 `http://localhost:5173`
- [ ] 打开浏览器开发者工具查看网络请求
- [ ] 检查 Console 中的错误信息

**常见原因：**
1. 后端服务未启动
2. 端口被占用
3. CORS 配置问题

### ❌ 错误：AI 无响应或响应异常

**检查清单：**
- [ ] DeepSeek API Key 是否有效
- [ ] 网络连接是否正常
- [ ] 查看 Conductor Agent 日志中的详细错误

**解决方法：**
检查 `agents/conductor_agent/agent.py` 中的 API 配置。
